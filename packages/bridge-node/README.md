# bridge node

> L1<>L2 Bridge node

## Getting started

Install dependencies:

```bash
npm install
```

Run node:

```bash
npm start
```

Run E2E example:

```bash
npm run example
```

Run arbitrum node:

```bash
npm run arbitrum:init -- 0xC34Fd04E698dB75f8381BFA7298e8Ae379bFDA71 https://kovan.infura.io/v3/[id]
npm run arbitrum:run -- 0xC34Fd04E698dB75f8381BFA7298e8Ae379bFDA71 --password=[password]
```
