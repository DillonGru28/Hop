# bridge node

> Bridge node scripts

## Getting started

Install dependencies:

```bas
npm install
```

Add committed private key to `secret.json`

```bash
{
  "committeePrivateKey": "<private-key>"
}
```

Run watcher 1:

```bash
npm run watcher1
```

Run watcher 2:

```bash
npm run watcher2
```

Run watcher 3:

```bash
npm run watcher3
```
