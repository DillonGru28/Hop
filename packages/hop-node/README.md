# Hop Node

> Hop node for bonding transfers

## Getting started

Run bridge node watcher:

```bash
docker run --env-file .env hopprotocol/hop-node:latest
```

Bridge node config:

```bash
BONDER_PRIVATE_KEY=0x111...
```

## Development

Install dependencies:

```bash
npm install
```

Run node:

```bash
npm start
```
